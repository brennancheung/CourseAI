# Session Logs

This directory contains Markdown files from post-exercise review conversations with Claude Code.

## File Naming

`YYYY-MM-DD-<exercise-slug>.md` or `YYYY-MM-DD-session.md` for general sessions.

## Purpose

These files capture:
- What happened during practice
- What worked and what was hard
- Curiosities and questions that emerged
- Insights for future lesson planning

## How Sessions Are Created

1. Complete an exercise in Cubase
2. Open Claude Code and run `/review`
3. Have a conversation about how it went
4. Claude Code saves notes to this directory

## Using Session History

Claude Code reads these files to:
- Reference previous struggles and progress
- Inform lesson planning and exercise design
- Track patterns over time
